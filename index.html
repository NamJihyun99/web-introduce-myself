02
장점

1. 자바

가장 오래 공부했고, 가장 많이 사용하는 언어입니다. 개발할 때마다
객체지향적인 코드에 관심이 많고 고민도 많습니다. 최근에는 프로그래밍에
관심이 많아서 함께 이야기 나눠도 좋을 것 같습니다.

2. 데이터베이스

프로젝트에서 주로 사용해 본 것은 관계형 데이터베이스이지만 NoSQL
데이터베이스에도 관심이 많습니다. 최근에는 관계형 데이터베이스의
인덱싱과 원리에관심이 많아져서 공부하고 있습니다.

03
장점

3. 스프링 프레임워크

자바가 가장 익숙한 언어였기 때문에 웹 백엔드 개발을 시작하면서 스프링
프레임워크를 공부하게 되었습니다.

4. 통계학 전공 지식

복수전공이 통계학이어서, 약간의 전공지식을 가지고 있습니다. 이론에 대한 공부를
주로 해와서 이번 기회를 통해 프로젝트에 활용해 볼 수 있으면 좋을 것 같습니다.

5. 알고리즘과 코딩테스트

엄청 뛰어난 실력을 가진 것은 아니지만 예전부터 꾸준히 공부하고 문제 풀이 연습을
해왔습니다.
